## Global Economic Indicator (Population & GDP) – Power BI Project

## Project Overview

This project presents an interactive data visualization of Global economic indicator based on:
- Population
- Gross Domestic Product (GDP)

The goal is to provide clear, insightful, and interactive dashboards to explore economic growth patterns across globe using Power BI.

## Objectives
- Visualize GDP growth accross the globe
- Count countries accross different region of the globe
- Find country wise population density and gdp per capita

## Tools & Technologies Used
- Power BI – Data visualization and dashboard creation
- GitHub – Version control and project sharing
- Dataset Sources – world bank

## Dashboard Features
### 1. Overview
- Map visal showing GDP across region
- Bar chart illustrates country count over the region
- Table chart with information like population density, gdp per capita focusing on year 2020
- It has drill through option to view more details of particular country
